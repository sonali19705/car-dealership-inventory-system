# Car Dealership Inventory Management System

## Overview

A full-stack web application for managing a car dealership's inventory. The system provides role-based access for administrators and customers, allowing inventory management, vehicle browsing, authentication, and purchase workflows.

This project is being developed using Test-Driven Development (TDD) practices as part of the Incubyte Software Craftsman Internship Assessment.

---

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- bcryptjs
- Jest
- Supertest

### Frontend
- React (Vite)
- Tailwind CSS
- React Router
- Axios
- Context API

## Getting Started

### Clone Repository

```bash
git clone <repository-url>
cd car-dealership-inventory
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
